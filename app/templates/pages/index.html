Zen of Python

Apaixonado por tecnologia e principalmente pela linguagem Python, estou sempre em busca por aprender cada vez mais, sempre respeitando as boas práticas e buscando escrever um código limpo, escalável e de fácil manutenção.

Sou um estusiasta na programação e já experimentei diversas tecnologias, mas dentre elas acabei me apegando no desenvolvimento web fullstack com ênfase no backend usando Python e Django com frameworks como Django Rest Framework e Django Ninja para construções de APIs. No frontend trabalho com TypeScript usando principalmente React ou Vue para construção da interface.
